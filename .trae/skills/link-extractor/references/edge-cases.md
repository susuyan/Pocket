# 边界处理

## 无法访问或已删除
输入: 无效 URL 或 404 页面  
输出:
```markdown
- [原始 URL](原始 URL) — ⚠️ 无法访问或已删除
```

## 无外部链接但有正文
输入: 社交媒体帖子无外链,但包含有价值观点  
处理:
- 提取核心观点或结论
- 归档到 🤓 Fav Finds 或 🪶 Remember This
输出:
```markdown
- [原始帖子](URL) shared by [作者]
- 核心观点总结
```

## 多条输入处理
输入: 一次提供多个链接  
处理:
- 每条链接独立处理
- 分别归档,不混淆主链接与相关链接
- 按提交顺序输出

## 同一 URL 重复出现
处理:
- 当日仅记录一次
- 更新时合并来源信息
- 保留最完整的描述
