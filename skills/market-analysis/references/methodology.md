# 方法论详解

## 三步框架

```
STEP 1: 大环境与自身公司 → STEP 2: 客户 → STEP 3: 竞争对手
```

## STEP 1: 大环境与自身公司分析

### PEST 分析详解

#### Political (政治因素)

| 子因素 | 分析方法 | 市场影响 |
|---|---|---|
| 政策法规 | 梳理相关政策文件 | 合规成本、准入门槛 |
| 政治稳定性 | 评估政治环境变化 | 投资风险、长期规划 |
| 贸易政策 | 关税、配额、自贸协定 | 成本结构、市场准入 |
| 产业政策 | 补贴、限制、扶持力度 | 竞争环境、发展机会 |

#### Economic (经济因素)

| 子因素 | 分析方法 | 市场影响 |
|---|---|---|
| 经济增长率 | GDP增长率、行业增速 | 市场规模、发展潜力 |
| 利率水平 | 央行政策、市场利率 | 融资成本、投资回报 |
| 汇率变动 | 汇率走势、波动幅度 | 进出口成本、定价策略 |
| 通胀水平 | CPI、PPI指标 | 成本压力、定价能力 |
| 收入水平 | 人均收入、可支配收入 | 消费能力、市场潜力 |

#### Social (社会因素)

| 子因素 | 分析方法 | 市场影响 |
|---|---|---|
| 人口结构 | 年龄分布、城镇化率 | 消费偏好、服务需求 |
| 教育水平 | 受教育程度、人才供给 | 产品复杂度、创新能力 |
| 生活方式 | 消费习惯、价值观 | 产品需求、营销方式 |
| 文化趋势 | 流行文化、亚文化 | 设计偏好、品牌认同 |

#### Technological (技术因素)

| 子因素 | 分析方法 | 市场影响 |
|---|---|---|
| 技术发展速度 | 技术迭代周期 | 产品生命周期 |
| 新技术应用 | 识别颠覆性技术 | 行业格局变化 |
| 技术壁垒 | 专利、技术难度 | 进入门槛、护城河 |
| 数字化程度 | 数字化渗透率 | 渠道、营销方式 |

### 五力分析详解

#### 1. 现有竞争者竞争强度

**评估维度：**
- 竞争者数量
- 市场集中度（CR3, CR5）
- 行业增长率
- 固定成本占比
- 退出壁垒

**高竞争迹象：**
- 竞争者众多
- 市场增长缓慢
- 产品同质化严重
- 价格战频繁

#### 2. 潜在进入者威胁

**进入壁垒类型：**
- 规模经济
- 品牌认知度
- 资金需求
- 分销渠道
- 政府许可
- 转换成本

**威胁评估：**
- 高壁垒 = 低威胁
- 低壁垒 = 高威胁

#### 3. 替代品威胁

**替代程度判断：**
- 替代品性价比
- 客户转换成本
- 客户对替代品的认知

**威胁评估：**
- 替代品多 = 高威胁
- 替代品少 = 低威胁

#### 4. 供应商议价能力

**高议价能力因素：**
- 供应商集中度高
- 替代供应商少
- 转换成本高
- 供应商前向整合可能性

**低议价能力因素：**
- 供应商分散
- 替代品多
- 供应商竞争激烈

#### 5. 客户议价能力

**高议价能力因素：**
- 客户集中度高（大客户）
- 标准化产品（替代品多）
- 客户信息充分
- 客户后向整合可能性

**低议价能力因素：**
- 客户分散
- 产品差异化
- 转换成本高

### VRIO 分析详解

#### Value (价值化)

**问题：** 资源/能力能否创造竞争优势？

**判断标准：**
- 降低成本
- 提高产品差异化
- 满足客户未满足需求

**结果：**
- 是 → 进入下一步
- 否 → 竞争劣势

#### Rarity (稀缺性)

**问题：** 这种资源/能力是否稀缺？

**判断标准：**
- 少数企业拥有
- 难以快速获得

**结果：**
- 是 → 进入下一步
- 否 → 竞争平价

#### Inimitability (难以模仿)

**问题：** 能否轻易被复制？

**模仿难度来源：**
- 独特的历史路径
- 因果模糊性
- 社会复杂性

**结果：**
- 是 → 进入下一步
- 否 → 暂时竞争优势

#### Organization (组织化)

**问题：** 企业能否有效组织利用这种资源/能力？

**判断标准：**
- 组织结构支持
- 管理流程完善
- 文化氛围匹配

**结果：**
- 是 → 持续竞争优势
- 否 → 未实现的潜在优势

### SWOT 分析详解

#### Strengths (优势) - 内部正面因素

**分析方向：**
- 资源优势：资金、人才、技术
- 能力优势：研发、营销、运营
- 市场优势：品牌、渠道、客户
- 成本优势：规模、效率、供应链

#### Weaknesses (劣势) - 内部负面因素

**分析方向：**
- 资源不足：资金短缺、人才缺乏
- 能力欠缺：技术落后、营销弱
- 市场问题：品牌弱、渠道少
- 成本问题：效率低、成本高

#### Opportunities (机会) - 外部正面因素

**分析方向：**
- 市场机会：新市场、增长领域
- 政策机会：政府支持、法规宽松
- 技术机会：新技术应用
- 合作机会：战略联盟、并购

#### Threats (威胁) - 外部负面因素

**分析方向：**
- 竞争威胁：新进入者、替代品
- 政策威胁：监管收紧、政策变化
- 技术威胁：颠覆性技术
- 经济威胁：经济下行、汇率变动

**SWOT 矩阵应用：**

| | 机会(O) | 威胁(T) |
|---|---|---|
| **优势(S)** | SO: 发挥优势抓住机会<br>(增长战略) | ST: 利用优势应对威胁<br>(多元化战略) |
| **劣势(W)** | WO: 改进劣势抓住机会<br>(扭转战略) | WT: 减少劣势规避威胁<br>(防御战略) |

## STEP 2: 客户分析详解

### 帕雷托分析详解

#### 帕雷托原则

```
80% 的结果来自 20% 的原因
80% 的收益来自 20% 的客户
80% 的问题来自 20% 的原因
```

#### 分析步骤

1. **数据收集**
   - 按收益列出所有客户
   - 收集客户收益、购买频次、消费金额数据

2. **排序分析**
   - 按收益降序排列
   - 计算累计收益百分比

3. **识别关键客户**
   - 前20%客户贡献率
   - 是否符合80/20规律

4. **策略制定**
   - 关键客户：VIP服务、深度合作
   - 潜力客户：培育、转化
   - 边缘客户：维护、优化成本

#### 应用场景

- 客户分层
- 资源分配
- 服务策略
- 营销精准度

### RFM 分析详解

#### 指标定义

| 指标 | 说明 | 数据来源 | 评分标准 |
|---|---|---|
| **R** Recency | 距离上次购买天数 | 交易记录 | 最近=高分，越远=低分 |
| **F** Frequency | 特定时期购买次数 | 交易记录 | 频次高=高分，频次低=低分 |
| **M** Monetary | 累计消费金额 | 交易记录 | 金额高=高分，金额低=低分 |

#### 评分方法（5分制）

```
R评分:
1-30天 = 5分
31-90天 = 4分
91-180天 = 3分
181-365天 = 2分
>365天 = 1分

F评分:
>10次 = 5分
7-10次 = 4分
4-6次 = 3分
2-3次 = 2分
1次 = 1分

M评分:
>1万 = 5分
5000-1万 = 4分
2000-5000 = 3分
500-2000 = 2分
<500 = 1分
```

#### 客户分类

| RFM分 | 比例 | 特征 | 策略 |
|---|---:|---|---|
| 555 | 5% | 顶尖客户，最近、频繁、高消费 | VIP维护、专属服务 |
| 5X5 | 10% | 高价值但频次下降 | 唤醒、提升复购 |
| 55X | 10% | 活跃但消费下降 | 推荐相关产品 |
| 111 | 20% | 流失风险 | 唤醒活动、挽回 |
| ... | ... | ... | ... |

### 人物誌

#### 创建方法

1. **数据收集**
   - 客户调研
   - 数据分析
   - 访谈记录
   - 行为数据

2. **模式识别**
   - 共同特征
   - 典型行为
   - 共同痛点

3. **人物归纳**
   - 选择1-3个代表性人物
   - 完善人物特征
   - 验证人物准确性

4. **应用推广**
   - 产品设计
   - 营销策略
   - 服务设计

### 同理心地图详解

#### 四个维度深入

**Say (说的)**
- 客户公开表达的内容
- 投诉、咨询、反馈
- 社交媒体言论

**Think (想的)**
- 客户内心考虑
- 未说出来的思考
- 担忧、期望、怀疑

**Do (做的)**
- 客户实际行为
- 使用方式、操作习惯
- 决策路径

**Feel (感受的)**
- 客户情绪状态
- 焦虑、兴奋、沮丧
- 对产品的情感反应

#### 分析框架

```
客户需求分析
├─ 做什么 (What)
│  └─ 产品/服务需求
├─ 为什么 (Why)
│  └─ 动机、目标
├─ 怎么做 (How)
│  └─ 使用行为路径
└─ 感受如何 (Feeling)
   └─ 情绪体验
```

### 客户体验旅程图详解

#### 旅程阶段

| 阶段 | 说明 | 客户目标 |
|---|---|---|
| 认知 | 发现产品存在 | 了解这是什么 |
| 考虑 | 评估产品价值 | 判断是否适合 |
| 购买 | 决策购买 | 完成交易 |
| 使用 | 使用产品 | 解决问题 |
| 留存 | 持续使用 | 获得价值 |
| 推荐 | 分享推荐 | 影响他人 |

#### 各层分析

**1. 触点**
- 客户接触产品的所有点
- 广告、官网、客服、产品界面等

**2. 客户行为**
- 每个阶段的具体行为
- 点击、浏览、购买、使用等

**3. 客户情绪**
- 每个阶段的情绪变化
- 好奇、疑虑、期待、满意等

**4. 客户想法**
- 每个阶段的内心想法
- 疑惑、担忧、满意、建议等

**5. 痛点**
- 每个阶段的困难和障碍
- 找不到、不会用、不满意等

**6. 机会**
- 可以改进的地方
- 优化体验、增加功能等

## STEP 3: 竞争对手分析详解

### 4P 分析详解

#### Product (产品)

| 分析维度 | 问题 |
|---|---|
| 功能性 | 有哪些功能？差异化在哪里？ |
| 质量 | 产品质量如何？可靠性？ |
| 设计 | 外观设计、用户体验 |
| 品牌 | 品牌定位、认知度 |
| 服务 | 售后服务、支持 |

#### Price (价格)

| 分析维度 | 问题 |
|---|---|
| 定价策略 | 高端/中端/低端市场？ |
| 价格结构 | 一次性收费/订阅/免费+增值？ |
| 折扣政策 | 有什么促销？ |
| 价值感知 | 客户认为值不值？ |

#### Place (渠道)

| 分析维度 | 问题 |
|---|---|
| 分销方式 | 线上/线下/直销/代理？ |
| 覆盖范围 | 哪些地区/市场？ |
| 渠道效率 | 渠道成本如何？ |
| 渠道控制 | 直接控制还是依赖合作？ |

#### Promotion (推广)

| 分析维度 | 问题 |
|---|---|
| 广告投入 | 投放哪些渠道？ |
| 营销内容 | 什么样的内容和主题？ |
| 活动策略 | 有什么营销活动？ |
| 销售促进 | 促销、打折、会员？ |

### 价值链分析详解

#### 价值链模型

```
                    [基础设施]
                          │
          ┌───────────────┼───────────────┐
          │               │               │
     [人力资源]      [技术开发]       [采购]
          │               │               │
          └───────────────┼───────────────┘
                          │
                      [主要活动]
                          │
    ┌─────┬─────┬─────┬─────┬─────┐
    │  A  │  B  │  C  │  D  │  E  │
┌───┴─────┴─────┴─────┴─────┴─────┴───┐
│ [进货] [生产] [出货] [市场] [服务]  │
│  物流   运营   物流   销售         │
└───────────────────────────────────┘
              边际
              利润
```

#### 主要活动

| 活动 | 说明 | 价值创造点 |
|---|---|---|
| 进货物流 | 接收、储存、分配原材料 | 成本控制、供应可靠性 |
| 生产运营 | 将原材料转化为产品 | 产品质量、效率 |
| 出货物流 | 分送成品给客户 | 速度、可靠性 |
| 市场销售 | 引导客户购买 | 市场份额、品牌 |
| 客户服务 | 售后支持 | 客户满意度、忠诚度 |

#### 辅助活动

| 活动 | 说明 | 价值创造点 |
|---|---|---|
| 基础设施 | 组织结构、管理、规划 | 整体效率、决策质量 |
| 人力资源 | 招聘、培训、激励 | 人才质量、能力 |
| 技术开发 | 研发、技术改进 | 创新、差异化 |
| 采购 | 购买原材料、设备 | 成本、质量 |

#### 分析应用

**价值链分析目的：**
1. 识别价值创造环节
2. 找出成本控制点
3. 发现差异化机会
4. 定位核心能力

**分析方法：**
- 每个活动的成本占比
- 每个活动对客户价值的贡献
- 与竞争对手的对比
- 优化和改进机会

### 核心能力分析详解

#### 核心能力特征

| 特征 | 说明 | 例证 |
|---|---|---|
| 稀缺性 | 少数企业拥有 | 专利技术、独家资源 |
| 难以模仿 | 竞争者难以复制 | 历史积累、复杂系统 |
| 客户价值 | 为客户创造价值 | 更好体验、更低成本 |
| 可延展性 | 可扩展到新领域 | 技术平台、品牌 |

#### 核心能力类型

| 类型 | 能力内容 | 持久性 |
|---|---|---|
| 技术能力 | 研发、专利、技术 | 高 |
| 市场能力 | 品牌、渠道、关系 | 中 |
| 运营能力 | 流程、效率、供应链 | 中 |
| 创新能力 | 敏捷、迭代、模式 | 高 |

#### 识别方法

1. **价值贡献测试**
   - 这种能力是否对客户价值有明显贡献？
   - 是否是企业成功的关键因素？

2. **稀缺性测试**
   - 竞争对手是否缺乏这种能力？
   - 是否难以快速获得？

3. **不可模仿性测试**
   - 竞争对手能否复制？
   - 复制需要多长时间和多高成本？

4. **组织化测试**
   - 企业是否充分利用了这种能力？
   - 是否有相应的组织支持？

#### 应用策略
