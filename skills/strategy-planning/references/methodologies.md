# 方法论详解

## STEP 1: 策略方向

### 1.1 產品組合矩陣 (BCG Matrix)

#### 起源
由波士頓諮詢公司（BCG）創始人Bruce Henderson於1970年提出。

#### 核心原理
基於兩個維度評估產品：
- **市場成長率**：衡量市場吸引力
- **市場份額**：衡量競爭地位

#### 四個象限

| 象限 | 市場成長率 | 市場份額 | 現金流 | 策略 |
|------|-----------|---------|--------|------|
| **明星產品** | 高 | 高 | 不穩定 | 投資維持 |
| **現金牛** | 低 | 高 | 正向 | 收穫現金 |
| **問題產品** | 高 | 低 | 負向 | 選擇性投資或退出 |
| **瘦狗** | 低 | 低 | 負向或零 | 退出或放棄 |

#### 策略應用

**明星產品**
- 持續投資維持市場領先地位
- 防守競爭者挑戰
- 適合做未來主要營收來源

**現金牛**
- 利用現金流投資其他產品
- 不需大量追加投資
- 延長生命週期，維持現金来源

**問題產品**
- 評估潛力，選擇性投資
- 投資後可轉為明星，或不投資轉為瘦狗
- 不宜投入過多資源

**瘦狗**
- 經營不善的舊產品
- 考慮退出或出售
- 避免投入資源

#### 使用步驟
1. 計算所有產品的相對市場份額
2. 剖析產品所在市場的成長率
3. 分配到適當象限
4. 制定各產品策略

#### 注意事項
- 市場份額需相對主要競爭者計算
- 高成長率通常定義為>10%
- 不考慮短期收益，注重長期價值

### 1.2 安索夫矩陣 (Ansoff Matrix)

#### 起源
由 Igor Ansoff 於1957年在「哈佛商業評論」提出。

#### 核心原理
基於市場和產品兩個維度，定義四種成長策略：

| | 現有市場 | 新市場 |
|---|---------|-------|
| **現有產品** | 市場滲透 | 市場開發 |
| **新產品** | 產品開發 | 多角化 |

#### 四種成長策略詳細說明

**市場滲透**
- 定義：在現有市場推廣現有產品
- 目標：提高現有客戶的購買頻率或購買量
- 策略：價格促銷、增加行銷、提高客戶滿意
- 風險：低

**市場開發**
- 定義：將現有產品推廣到新市場
- 目標：拓展新地區、新客群、新區隔
- 策略：區域擴展、國際化、新客群定位
- 風險：中等

**產品開發**
- 定義：為現有市場開發新產品
- 目標：滿足現有客戶的新需求
- 策略：產品升級、功能延伸、相關產品
- 風險：中等

**多角化**
- 定義：進入新市場、推出新產品
- 目標：高成長、分散風險
- 策略：相關多角化、不相關多角化
- 風險：高

#### 風險比較

```
低風險  ━━━━━━━━━━━━━  高風險
   A   │
       │      D  C
       │
       │   B
━━━━━━━━━━━━━━━━━━━━━━━
       │   低風險  高風險
```

**風險排序**：
1. 市場滲透 (最低風險)
2. 市場開發 / 產品開發 (中等風險)
3. 多角化 (最高風險)

#### 使用場景
- 制定企業成長策略
- 評估新業務投資機會
- 規劃國際化或產品線擴展

### 1.3 交叉SWOT

#### 起源
由 Albert Humphrey 於 1960-70 年代於史丹福大學發展。

#### 基礎分析

**內部分析**：
- **Strengths（優勢）**：企業內部的優勢能力
- **Weaknesses（劣勢）**：企業內部的不足之處

**外部分析**：
- **Opportunities（機會）**：外部環境的有利因素
- **Threats（威脅）**：外部環境的不利因素

#### 交叉策略矩陣

| | 優勢 (S) | 劣勢 (W) |
|---|---------|---------|
| **機會 (O)** | **SO策略**<br>利用優勢把握機會 | **WO策略**<br>克服劣勢把握機會 |
| **威脅 (T)** | **ST策略**<br>利用優勢抵禦威脅 | **WT策略**<br>減少劣勢躲避威脅 |

#### 四種策略類型

**SO策略（優勢-機會）**
- 利用內部優勢抓住外部機會
- 進攻型策略
- 舉例：利用技術優勢進入快速成長市場

**WO策略（劣勢-機會）**
- 克服內部劣勢抓住外部機會
- 改進型策略
- 舉例：引進技術合作以進入新市場

**ST策略（優勢-威脅）**
- 利用內部優勢應對外部威脅
- 防禦型策略
- 舉例：利用品牌優勢應對價格戰

**WT策略（劣勢-威脅）**
- 內外部都不利
- 收縮型策略
- 舉例：退出不具優勢的市場

#### 分析步驟
1. 列出內部優勢和劣勢
2. 列出外部機會和威脅
3. 交叉匹配生成策略選項
4. 優先選擇SO策略，再考慮其他

### 1.4 STP分析

#### 起源
由菲利普·科特勒（Philip Kotler）推廣的現代市場營銷核心框架。

#### 三個環節

**S - Segmentation（市場細分）**

將廣大市場視為由多個子市場組成，每個子市場有不同的需求。

**細分變數**：
| 類型 | 變數 | 範例 |
|------|------|------|
| 人口 | 年齡、性別、收入、職業 | 18-30歲女性都市專業 |
| 地理 | 地區、城市規模、氣候 | 北部、熱帶地區 |
| 心理 | 個性、生活方式、價值觀 | 健康意識高者 |
| 行為 | 使用情境、品牌忠誠、使用率 | 每日使用者 |

**有效細分標準**：
- 可衡量：細分市場規模可量化
- 可達成：有效觸及細分市場
- 足夠大：具營利潛力
- 可發揮：能發展針對性的策略

**T - Targeting（目標市場選擇）**

評估並選擇目標市場。

**選擇策略**：
| 策略 | 說明 | 適用場景 |
|------|------|---------|
| 無差異化 | 一種產品服務所有市場 | 產品標準化 |
| 差異化 | 多種產品服務不同市場 | 資源充足 |
| 集中化 | 專注單一細分市場 | 資源有限 |

**評估標準**：
- 市場規模：潛在營收
- 成長率：未來發展
- 競爭強度：競爭者多寡與實力
- 企業匹配度：能力是否匹配

**P - Positioning（定位）**

在目標客戶心目中建立獨特地位。

**定位維度**：
- 功能定位：功能層面的優勢
- 情感定位：情感層面的連結
- 品質定位：製程、設計
- 價格定位：高價/平價

**定位策略**：
| 策略 | 說明 | 範例 |
|------|------|------|
| 屬性定位 | 基於產品屬性 | 「最快的手機」 |
| 利益定位 | 基於客戶利益 | 「最省錢的方案」 |
| 用戶定位 | 基於用戶類型 | 「年輕人的選擇」 |
| 競爭定位 | 相對競爭者 | 「比X還好」 |
| 質量/價格定位 | 高品質/低價格 | 「物超所值」 |

### 1.5 定位圖 (Positioning Map)

#### 核心原理
以兩個關鍵維度繪製市場上所有競爭者的相對位置，找出市場機會。

#### 創建步驟
1. 選擇兩個關鍵競爭維度
2. 將競爭者標示於圖上
3. 分析位置與市場空缺
4. 決定企業定位

#### 維度選擇原則
- 與該產品類別相關
- 客戶重視的因素
- 可量化或可觀察
- 區分力強

#### 範例：咖啡店競爭定位

```
         價格
    高 │   星巴克       Blue Bottle
       │
       │           肯德基咖啡
       │ McCafe
       │
       │       85度C
    低 │   7-ELEVEN
       └─────────────────────
             品質
    低                 高
```

#### 分析應用

**市場空缺**：
- 高價格低品質：一般不建議
- 低價格高品質：理想定位但難達成
- 高價格高品質：精品定位
- 低價格低品質：實用定位

**定位策略**：
1. **接近競爭者**：直接競爭，需明確差異
2. **定位中間**：尋求平均市場
3. **定位角落**：利基市場

## STEP 2: 實現方式

### 2.1 商業模式圖 (Business Model Canvas)

#### 起源
由 Alexander Osterwalder 於2008年提出。

#### 九大構塊

**客戶層面**：

| 構塊 | 說明 | 舉例 |
|------|------|------|
| 客戶細分 | 我們的客戶是誰？ | 商務人士、健康意識者 |
| 價值主張 | 我們提供什麼價值？ | 節省時間、提升效率 |
| 管道通路 | 我們如何觸達客戶？ | 線上、實體、合作夥伴 |
| 客戶關係 | 我們如何維持客戶？ | 自助服務、專人服務 |
| 收入來源 | 我們如何賺錢？ | 訂閱、單次購買 |

**企業層面**：

| 構塊 | 說明 | 舉例 |
|------|------|------|
| 核心資源 | 我們擁有什麼資源？ | 專利、團隊、資金 |
| 關鍵活動 | 我們做什麼？ | 生產、行銷、客服 |
| 關鍵合作 | 我們與誰合作？ | 供應商、合作伙伴 |
| 成本結構 | 我們花費什麼？ | 羅資、營運、行銷 |

#### 應用方法
1. 列出九大構塊的內容
2. 檢查各構塊間的一致性
3. 識別商業模式優化機會

### 2.2 流程圖 (Process Flowchart)

#### 基本符號

| 符號 | 含義 | 用途 |
|------|------|------|
| 橢圓 | 開始/結束 | 流程起點和終點 |
| 矩形 | 流程步驟 | 主要工作項目 |
| 菱形 | 決策點 | yes/no branching |
| 平行線 | 不連續 | 跳過的步驟 |

#### 建圖步驟
1. 定義流程範圍和目標
2. 列出所有步驟
3. 添加決策點
4. 标注角色和資源
5. 識別瓶頸

### 2.3 AIDMA / AISAS

#### AIDMA 模型（傳用）

| 階段 | 英文 | 中文 | 目標 |
|------|------|------|------|
| A | Attention | 注意 | 引起客戶關注 |
| I | Interest | 興趣 | 激發客戶興趣 |
| D | Desire | 欲望 | 建立購買慾望 |
| M | Memory | 記憶 | 品牌記憶 |
| A | Action | 行動 | 購買行為 |

#### AISAS 模型（數位時代）

| 階段 | 英文 | 中文 | 差異 |
|------|------|------|------|
| A | Attention | 注意 | 引起關注 |
| I | Interest | 興趣 | 激發興趣 |
| S | Search | 搜尋 | **新階段**：主動查找資訊 |
| A | Action | 行動 | 下單購買 |
| S | Share | 分享 | **新階段**：分享體驗 |

#### 行銷漏斗應用

```
注意: 100%
   ↓ 80%
興趣: 80%
   ↓ 60%
搜尋: 48%
   ↓ 50%
行動: 24%
   ↓ 30%
分享: 7%
```

#### 優化策略

**注意階段**：
- 強化廣告曝光
- 社群媒體行銷

**興趣階段**：
- 提供有價值內容
- 展示產品特色

**搜尋階段**：
- SEO優化
- 收集客戶評價

**行動階段**：
- 簡化購買流程
- 促銷優惠

**分享階段**：
- 分享激勵
- 產品特點便於分享

### 2.4 甘特圖 (Gantt Chart)

#### 核心原理
以時間為橫軸，任務為縱軸，顯示專案進度與關聯。

#### 結構要素

| 欄位 | 說明 |
|------|------|
| 任務名稱 | 工作項的標識 |
| 開始日期 | 計畫開始時間 |
| 結束日期 | 計畫完成時間 |
| 持續時間 | 工作天數計算 |
| 負責人 | 執行此任務的人員 |
| 前置任務 | 依賴的其他任務 |
| 里程碑 | 關鍵時點 |

#### 建圖步驟
1. 列出所有任務
2. 確定任務順序和關聯
3. 估計時間
4. 分配資源
5. 標注里程碑

### 2.5 組織圖 (Organization Chart)

#### 主要類型

**功能型部門**
```
CEO
 ├─ 行銷部
 ├─ 技術部
 ├─ 財務部
 └─ 人資部
```
特點：按職能分工，權責明確，適用穩定業務。

**專案型組織**
```
CEO
 ├─ 專案A
 │   ├─ 行銷成員
 │   └─ 技術成員
 └─ 專案B
     ├─ 行銷成員
     └─ 技術成員
```
特點：專案導向，彈性高，適用專案制工作。

**矩陣式組織**
```
CEO
 ├─ 專案A ─┬─ 行銷部
 │        └─ 技術部
 └─ 專案B ┬─ 財務部
          └─ 人資部
```
特點：結合功能與專案，適用複雜組織。
