# 决策树

## 选择主链接
```
正文中是否包含 GitHub 仓库链接?
├─ 是 → 取首个 GitHub 仓库
└─ 否 → 是否有官方文档/项目主页?
       ├─ 是 → 取官方文档/主页
       └─ 否 → 是否有产品/工具页面?
              ├─ 是 → 取产品页面
              └─ 否 → 是否有明确标题的技术文章?
                     ├─ 是 → 取技术文章
                     └─ 否 → 取正文中被强调的链接
```

## 选择输出样式
```
是否为 GitHub 仓库?
├─ 是 → GitHub Repo 样式
└─ 否 → 是否为官方博客?
       ├─ 是 → 官方博客样式
       └─ 否 → 是否为长文章(需快速判断)?
              ├─ 是 → 要点式摘要
              └─ 否 → 工具名称是否自解释?
                     ├─ 是 → 图标单行
                     └─ 否 → 概念式(两行)
```
