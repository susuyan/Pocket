# 工作流程

## 完整处理流程
```
1. 接收 URL
   ↓
2. 识别平台类型(Weibo/Twitter/HN/Blog/GitHub)
   ↓
3. 抓取页面内容
   - 提取标题、描述、正文
   - 收集所有外部链接
   - 展开短链接
   ↓
4. 智能分析
   - 按优先级选择主链接
   - 过滤无关资源(统计/头像/广告)
   - 识别关键词和相关链接
   ↓
5. 生成 Markdown 条目
   - 选择合适的样式(概念/要点/官方/Repo/图标)
   - 生成简短描述(≤120 字符)
   - 组装引用链接
   ↓
6. 归档到 Daily Links
   - 自动分类到六大板块之一
   - 检查去重
   - 写入对应日期文件
```
