# CLAUDE.md

Claude Code 工作指南

## 项目概述

Pocket 是 Claude Code 技能系统开发仓库，用于定义和管理 AI 辅助工作技能。

## 目录结构

```
Pocket/
├── skills/              # 技能定义（实际目录）
│   └── <skill-name>/
│       ├── SKILL.md
│       └── references/
├── .trae/skills/        # 软链接（Claude 读取位置）
├── docs/                # 文档归档
│   ├── daily-links/
│   └── articles/
└── scripts/             # 自动化脚本
```
